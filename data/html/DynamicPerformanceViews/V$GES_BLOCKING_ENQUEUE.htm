7.235 V$GES_BLOCKING_ENQUEUE

V$GES_BLOCKING_ENQUEUE describes all locks currently known to lock manager that are being blocked or blocking others. The output of this view is a subset of the output from V$GES_ENQUEUE. This is an Oracle Real Application Clusters view.

See Also:

"V$GES_ENQUEUE" for a description of all locks known to the lock manager
