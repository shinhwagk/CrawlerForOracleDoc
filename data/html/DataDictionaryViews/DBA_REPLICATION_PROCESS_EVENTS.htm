5.276 DBA_REPLICATION_PROCESS_EVENTS

DBA_REPLICATION_PROCESS_EVENTS provides information about the replication processes events in the database. Its columns are the same as those in ALL_REPLICATION_PROCESS_EVENTS.

Note:

This view is available starting with Oracle Database 12c Release 2 (12.2.0.1).
